# MACHINE LEARNING PROJECTS

This repository contains multiple Machine Learning projects developed using Python and Jupyter Notebook.

## 📂 Included Projects

### 1. Credit Card Dataset Analysis
- Data preprocessing and cleaning
- Exploratory Data Analysis (EDA)
- Machine Learning model implementation
- Prediction and evaluation

### 2. Food Delivery Time Prediction
- Delivery time analysis
- Feature engineering
- Prediction using Machine Learning algorithms
- Performance evaluation

### 3. KC House Price Prediction
- House price analysis using KC House dataset
- Data visualization and preprocessing
- Regression model building
- Price prediction and accuracy checking

## 🛠️ Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Scikit-learn
- Jupyter Notebook

## 📁 Files Included
- `credit_card_dataset.ipynb`
- `food_delivery_time.ipynb`
- `kc_house_data.ipynb`
- Dataset files (.csv / .xlsx)

## 🎯 Objective
The main objective of this repository is to practice and implement Machine Learning concepts using real-world datasets.
